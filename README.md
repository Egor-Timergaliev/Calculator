Техническое задание на разработку программы “Калькулятор”
Целью является создание программы на языке C#, которая выполняет базовые арифметические операции: сложение, вычитание, умножение, деление, а также возведение числа в степень и вычисление квадратного корня. 
Программа должна уметь: 
1. Принимать от пользователя два числа (для операций сложения, вычитания, умножения, деления и возведения в степень).
2. Выполнять выбранную пользователем операцию.
3. Показывать результат вычисления на экране.
4. Для вычисления корня — принимать одно число и выводить результат.
5. Обрабатывать ошибки, например, деление на ноль или ввод нечисловых данных. 
Тест-кейсы:
1. Входные данные:	a = 10, b = 5; Операция: Деление;	Ожидаемый результат: 2
2. Входные данные:	a = 4, b = 2; Операция: Возведение в степень;	Ожидаемый результат: 16
3. Входные данные:	a = 9; Операция: Квадратный корень;	Ожидаемый результат: 3
